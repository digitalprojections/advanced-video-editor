# Gemini Resolve - Advanced Video Editor

Professional-grade non-linear video editing workspace built with React.

## Description
Gemini Resolve is a powerful browser-based video editor that features a multi-page workspace including Media management, Cut, Edit, Audio, and Delivery pages. It supports timeline-based editing, real-time playhead driving, and frame capturing.

## Key Features
- **Media Pool**: Manage and organize your video assets.
- **Edit Page**: Multi-layer timeline with drag-and-drop clip placement.
- **Audio Workstation**: Dedicated interface for sound management.
- **Snapshot Engine**: Capture high-quality frames directly from the timeline.
- **GPU Acceleration**: High-performance rendering and playback.

## Installation & Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Launch Dev Server**:
   ```bash
   npm run dev
   ```

3. **Open the App**:
   Navigate to `http://localhost:3000/video-editor/` in your browser.
